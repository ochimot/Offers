how to use different IP AT THE SAME TIME
EXAMPLE:

1. right click of mouse on IP, chose 'Set Dedicated socks port' and one of 3 options (auto, manually, from fixed slot)

1.1 if manually, enter any port up to 65365

2. as you can see - IP 174.119.19.17 now match 127.0.0.1:3333 (port 3333 we set on previous step)

3. ok. now set other port to other IP

4. as you can see - IP 96.51.45.44 now match 127.0.0.1:5555 (port 5555 - it can be different, your choice)


now - you can set socks 127.0.0.1:5555 for IP 96.51.45.44
socks 127.0.0.1:3333 for IP 174.119.19.17 (in other programm)
and socks 127.0.0.1:9951 (default port for default socks)
AT THE SAME TIME
you can use up to 15 different socks on different ports AT THE SAME TIME

FOR EXAMPLE ONLY!!! YOUR IP WILL BE DIFFERENT
