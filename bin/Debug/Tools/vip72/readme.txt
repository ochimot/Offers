if you have used installer, just open socks client and proxifier from desktop, login, select IP and work with mozilla 

Comman line parameters:
-login;       set login and password from command line (example: -login;USERNAME:PASSWORD; ) 
-s5r:  		  connect client through external socks5 server, resolve DNS on socks side (example: vip72socks.exe -s5r:192.168.64.1:9999)
-s5:  		  connect client through external socks5 server, resolve DNS on client side
-copyip  	  copy IP to clipboard every time, when add new to "my proxies"
-nom		  disable context menu styles
-skin  		  enable window skin in WinXP
-ste		  strong traffic encryptoion on start (you can set it manually in context menu in tray)
-reconnect    auto-reconnect mode
-hidepass     hide password
-mp:          set CUSTOM default local port for scoks5 instead of 9951 (example: -mp:18018 )

РУССКИЙ

если вы использовали инсталлятор, то запустите сокс клиент, запустите проксифер с рабочего стола, выберите IP - этого должно быть достаточно для работы в mozilla firefox

Параметры коммандной строки:
-login;       ввод логина и пароля с коммандной строки (пример: -login;USERNAME:PASSWORD; ) 
-s5r:  		  подключение клиента через внешний сокс5 сервер, DNS на стороне сокса (пример: vip72socks.exe -s5r:192.168.64.1:9999)
-s5:  		  подключение клиента через внешний сокс5 сервер, DNS на стороне клиента
-copyip  	  копировать IP в буффер обмена каждый раз при добавлении нового IP
-nom		  отключить стили контекстного меню
-skin  		  включить "ШКУРУ" в WinXP
-ste		  strong traffic encryptoion при запуске (вы можете включить этот режим вручную из контекстного меню)
-reconnect    режим автопереподключения при обрыве связи
-hidepass     скрыть пароль при вводе
-mp:          установить изменённый порт по умолчанию для scoks5 вместо 9951(пример: -mp:18018 )
