как использовать разные IP ОДНОВРЕМЕННО
ПРИМЕР:

1. кликаем правой кнопкой мыши по IP, выбираем 'Set Dedicated socks port' и далее три опции на выбор (auto, manually, from fixed slot)

1.1 если manually, вводжим вручную любой порт до 65365

2. как можно видеть - IP 174.119.19.17 теперь соответствует локальному соксу 127.0.0.1:3333 (порт 3333 был введён на предыдущем шаге)

3. ok. теперь тоже самое для другого IP на другой порт

4. как можно видеть - IP 96.51.45.44 теперь соответствует локальному соксу 127.0.0.1:5555 (порт 5555 - может быть любой доступный по вашему выбору)


теперь - можно использовать сокс 127.0.0.1:5555 для IP 96.51.45.44
сокс 127.0.0.1:3333 для IP 174.119.19.17 (например в другой программе)
и сокс 127.0.0.1:9951 ( текущий сокс с портом по умолчанию 9951)
ОДНОВРЕМЕННО
вы можете использовать до 15 разных! соксов на разных портах ОДНОВРЕМЕННО

в качестве ip сокса может быть также (вместо 127.0.0.1) ip адрес любого локального интерфейса, в том числе виртуальной сети vmware
